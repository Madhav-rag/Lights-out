# Lights Out

A Formula 1 themed Day & Night experience built with Vite, HTML, CSS and JavaScript.

## Features
- F1 start lights intro sequence
- Day/Night theme toggle with scroll-driven transitions
- Monaco (Day) → Singapore (Night) atmosphere journey
- Live Sydney time HUD
- Engine start sound + F1 car flyby
- Information on certain F1 tracks as well.
