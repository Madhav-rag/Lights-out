# Lights Out

A Formula 1 themed Day & Night experience built with HTML, CSS and JavaScript.
Scroll from top to bottom of website to see transition between day and night on a
F1 race calendar with Monaco being the day race and Singapore/Saudi Arabia being the 
night race. Hope you enjoy navigating through my website, thanks!

## Features
- F1 start lights intro sequence
- Day/Night theme toggle with scroll-driven transitions
- Monaco (Day) → Singapore (Night) atmosphere journey
- Live Sydney time 
- Engine start sound + F1 car flyby
- Information on certain F1 tracks as well.
